# Kickstart-HTML

## Installation

Make sure your system is configured with [Nodejs], [NPM] and [Gulp]

### Configuration

```
$ npm install
```
```
$ bower install
```
```
$ Clone [Gulp Less Watcher] to the application root
```

### Run the Application
```
$ nodemon app.js
```

Now you add play with the view files
